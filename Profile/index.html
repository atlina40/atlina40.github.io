Project 3 — Return Forecasting: ARIMA vs LSTM

NVIDIA (NVDA) — ARIMA vs LSTM Return Forecasting

Compared a classical ARIMA benchmark against a deep-learning LSTM network for forecasting NVIDIA daily log returns. Used chronological 70/15/15 train-validation-test split with no data leakage. ARIMA(1,0,0) was selected as the recommended model due to comparable RMSE and superior directional accuracy, confirming that additional LSTM complexity is not justified for this highly efficient large-cap equity.
